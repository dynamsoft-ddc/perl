Introduction
The sample demonstrates how to convert image to text by using DDC Ocr Restful Api.

How to run the sample:
1. Before running the sample, please install the sample dependencies using CPAN. 
   cpan File::Slurp
   cpan File::Basename
   cpan File::Spec
   cpan JSON
   cpan HTTP::Request::Common
   cpan LWP::UserAgent
   cpan LWP::Protocol::https
   cpan Mozilla::CA
2. Update apiKey in config.json with your own secret key. For details on how to get a secret key, please check
   https://developer.dynamsoft.com/authenticating-to-the-cloud-api
3. Run the sample from the command line use the perl command:
   perl Program.pl
